# Contributing to Open Law Data Thailand

Thank you for your interest in contributing to the Open Law Data Thailand project! We welcome contributions from the community to help make Thai legal data more accessible and useful for everyone.

If you have built something using Open Law Data, please list it in our showcase!

1.  **Fork** this repository.
2.  Edit `showcases.toml` in the public/data directory.
3.  Add your project block to the `[[projects]]` list (append it to the end or appropriate position):

    ```toml
    [[projects]]
    title = "My Awesome Project"
    author = "Your Name"
    author_url = "https://github.com/yourname"
    description = "Short description of what it does (max 200 chars)."
    tags = ["AI", "Search", "Visualization"]
    url = "https://your-project-url.com"
    featured = false
    date = "YYYY-MM-DD"
    ```

    *   **title**: Name of your project.
    *   **author**: Your name or organization.
    *   **author_url**: Link to your GitHub profile or website.
    *   **description**: Brief explanation of your project.
    *   **tags**: List of relevant tags (e.g., "AI", "Web", "Mobile").
    *   **url**: Link to the project repo or live demo.
    *   **featured**: Set to `false` (admin will review and update if applicable).
    *   **date**: Date of submission or release (YYYY-MM-DD).

4.  Submit a **Pull Request**.

We look forward to seeing what you build!
